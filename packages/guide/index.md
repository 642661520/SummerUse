# 入门

SummerUse 是一个基于 pnpm workspace 管理的 monorepo，包含了多个子包。

- `@summeruse/ol`：用于 OpenLayers 的工具库。

- `@summeruse/cesium`：用于 Cesium 的工具库。

- `@summeruse/common`：公共工具库。

## 安装

以`@summeruse/ol` 为例，执行以下命令进行安装。

```bash
pnpm add @summeruse/ol ol
```
