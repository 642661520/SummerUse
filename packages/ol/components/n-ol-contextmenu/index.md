# n-ol-contextmenu

## 组件介绍

n-ol-contextmenu 组件是一个用于在 OpenLayers 地图上显示右键菜单的组件。它可以在地图上的任何位置显示一个菜单，当用户右键单击地图时，菜单将显示在鼠标位置。

::: tip 依赖
[`navie-ui`](https://www.naiveui.com/)
:::
